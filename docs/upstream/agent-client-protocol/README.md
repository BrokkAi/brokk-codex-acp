# Agent Client Protocol Upstream Docs

This directory contains a snapshot of the upstream Agent Client Protocol
documentation used by this adapter.

- Source index: `https://agentclientprotocol.com/llms.txt`
- Protocol docs: `protocol/v1/*.md`
- Canonical schema: `schema.json`
- Schema source: `https://github.com/agentclientprotocol/agent-client-protocol/releases/latest/download/schema.json`
- Retrieved: 2026-06-19

The `llms.txt` index also advertised
`https://agentclientprotocol.com/api-reference/openapi.json`, but that URL
returned `Asset not found` when this snapshot was taken. Use `schema.json` as
the protocol schema source of truth.
